# balística

## About
balística is a simple open source ballistics calculator. It's 
named after the Catalan word for "ballistics" in English. Enjoy!

## Minimum Requirements
* vala 0.18.0
* waf 1.6.11

## Build Instructions
```bash
cd balistica
waf configure
waf build
waf install
```
