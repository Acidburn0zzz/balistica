Welcome

Balística is a exterior balisitics calculator with some extra feature tacked on. The program is named for the Catalan word for "ballistics".

Currently the project is in it's very early stages and is still missing quite a few features I've planned for it. Feel free to jump in and help!

Current Features

Miller Twist Rule

The Miller Twist Rule is a mathematical formula dervised to help people calculate which bullets will work better in thier rifle based upon various factors. Including bullet length and barrel twist rate.

Greenhill Forumlua

The Greenhill Formula, in the same vien as the Miller Twist Rule is a much older formula for the same purpose that has been used for centuries now. It is the defacto standard for this type of computation.

Planned Features

A front end (GUI)

I really want a GUI front end. I'd like to have a layout that promotes ease of use and can be learned quickly without having to write RTFM everywhere.

A database to store custom handloads in

I want to be able to create and store custom hand loads. I already have a SQL database strucutre worked out for this. I also have some basic data to load in it. But that's all I have. I don't even have Vala to talk to the database with.

Calculate using the standard drag functions G1-G8

The standard drag functions are an essential part of any wana-be ballistics program. It's a have-to-have.

Seperate actual ballistics software into library

I'd like to have a LibBalistica so that people could theoretically use the same math in other programs. Or at the very least expand upon this one with little trouble.
